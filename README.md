# mycd

## C++ code:
	to build c++ code 
	cd cpp
	make clean
	make mycd
	
	binary can be find in bin folder
	
	to run the unit test case 
	cd cpp
	make clean
	make test
	
	 mkdir -p /abc/def/ghi
	 mkdir -p /abc/gh
	 mkdir -p /abc/klm
	 
	 ./test
	 
## golang code:
	to build golang code
	cd go

	binary can be find in bin folder
	
	to run the unit test case 
	cd go
